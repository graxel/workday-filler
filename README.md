# workday-filler

### Setup
1. Make sure you have UV set up.
2. Run `uv sync --all-groups`
3. Update `wdf.py` with your own information
4. Replace `resume.pdf` and `resume.json` with your own files.

### Use
1. Find a workday job you're interested in applying to.
2. Copy the URL.
3. Open `page_filler.ipynb`
4. Run the first cell, a browser opens.
5. Paste the url into the browser, click apply manually.
6. Run cell 2 to log in or create an account.
7. Run cell 3 to fill out your information.
8. Run cell 4 to fill out your work experience and education.
